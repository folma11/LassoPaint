# Lasso Paint

Lasso Paint is a production-ready Adobe Photoshop UXP plugin project skeleton for future implementation of automatic lasso fill workflows.

## Project goals

- Provide a clean Adobe Photoshop UXP plugin foundation.
- Keep the architecture simple and easy to extend.
- Prepare the codebase for future automation features.

## Structure

- `manifest.json` defines the Photoshop UXP panel entry point.
- `index.html` hosts the plugin UI.
- `js/` contains modular JavaScript for Photoshop integration, BatchPlay prep, UI rendering, and settings.
- `css/` contains panel styling.
- `docs/` holds architecture and implementation notes.

## Getting started

1. Open the folder in Adobe UXP Developer Tool.
2. Load the plugin by selecting the project root.
3. Ensure Photoshop 27.7 or newer is available.

## Notes

No feature implementation has been added yet. The current code is a clean scaffold with comments and module boundaries for future development.
